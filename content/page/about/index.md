## 소개

Flutter + Firebase 기반 학교 앱을 1인 개발하고 있는 개발자입니다.

이 블로그에서는 개발 과정에서 겪은 문제 해결기, 기술 선택의 이유, 그리고 다양한 기술 주제를 다룹니다.

## 기술 스택

- **Mobile**: Flutter, Dart, Riverpod
- **Backend**: Firebase (Firestore, Functions, FCM, Auth)
- **Web**: Next.js, React, Tailwind CSS
- **CI/CD**: GitHub Actions, Codecov
- **Tools**: Git, VS Code, Claude Code
